# Password Manager

A simple and secure **Password Manager** web application built with **React**, **Node.js**, and **MongoDB**. Store, view, copy, edit, and delete your passwords easily.

---

## Features

- Add passwords with **site**, **username**, and **password**
- View all saved passwords in a table
- Copy password, username, or site to clipboard
- Edit or delete existing passwords
- Show/Hide password functionality
- Responsive UI with React and Tailwind CSS
- Notifications using React Toastify

---

## Technologies Used

- **Frontend:** React, Tailwind CSS, React Toastify
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Others:** UUID for unique IDs, CORS for cross-origin requests
