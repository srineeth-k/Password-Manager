# 🔐 Password Manager

A secure full-stack Password Manager built using the **MERN stack (MongoDB, Express, React, Node.js)**.  
It allows users to safely store, manage, and access their credentials in a simple and secure way.

---

## 🚀 Features

- Add website, username, and password securely  
- View all saved credentials in a structured table  
- Copy site, username, or password with one click  
- Edit and delete stored entries  
- Show/Hide password toggle for security  
- Responsive UI built with React and Tailwind CSS  
- Real-time notifications using React Toastify  

---

## 🛡️ Security & Authentication

- Implemented **multi-layer security system** using **AES-256-GCM encryption** for storing sensitive data  
- Used **bcrypt hashing with salt** for secure password protection  
- Integrated **JWT-based authentication** for stateless and secure user sessions  
- Added **OAuth 2.0 login (Google & GitHub)** for seamless and secure third-party authentication  
- Built middleware-based authorization to ensure **user-specific data access control**

---

## ⚙️ Backend & API

- Designed a **RESTful API architecture** supporting full CRUD operations  
- Implemented custom middleware for **route protection and user data isolation**  
- Added centralized error handling for better debugging and stability  
- Configured **CORS policies** for secure frontend-backend communication  

---

## 🎨 Frontend

- Built a modern and responsive UI using **React and Tailwind CSS**  
- Managed global state using **Context API** for smooth session handling  
- Added real-time feedback using **React Toastify**  
- Implemented clean and intuitive user experience for password management  

---

## 🛠️ Tech Stack

- Frontend: React, Tailwind CSS  
- Backend: Node.js, Express  
- Database: MongoDB  
- Auth: JWT, OAuth 2.0  
- Security: bcrypt, AES encryption
